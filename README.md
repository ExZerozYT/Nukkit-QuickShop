# Nukkit-QuickShop
QuickShop for Nukkit

## How to use?

- **Install plugin**
  1. Download 'QuickShop' plugin.
  2. Put the plugin in the "plugins" folder.
  3. Restart your server.
  
## Config:(No changes are recommended)
